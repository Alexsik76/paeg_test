# Fast Q&A Search System (Система швидкого пошуку Q&A)

Веб-додаток доступний за адресою: [paeg-test.lab.vn.ua](http://paeg-test.lab.vn.ua)

Веб-додаток на базі Streamlit для швидкого пошуку та перегляду відповідей на поширені запитання (Q&A).
 Система підтримує формули LaTeX та використовує високопродуктивну бібліотеку Polars для обробки даних.

## 🛠 Технічний стек
- **Python 3.10+**
- **Streamlit**: Інтерфейс користувача.
- **Polars**: Швидка обробка та зчитування CSV даних.
- **PyYAML**: Керування конфігурацією через `config.yaml`.
- **KaTeX**: Рендеринг математичних формул (вбудовано в Streamlit).

## 🚀 Локальний запуск

### 1. Створення віртуального середовища
```bash
python -m venv venv
```

### 2. Активація середовища
- **Windows**:
  ```bash
  .\venv\Scripts\activate
  ```
- **Linux/macOS**:
  ```bash
  source venv/bin/activate
  ```

### 3. Встановлення залежностей
```bash
pip install -r requirements.txt
```

### 4. Запуск додатка
```bash
streamlit run main.py
```

## 📊 Алгоритм корегування даних (data/test.csv)

Якщо вам потрібно оновити список питань або відповідей, дотримуйтесь цього алгоритму:

1. **Завантаження/Копіювання**: Переконайтеся, що файл `data/test.csv` доступний на вашому комп'ютері.
2. **Відкриття в Excel**:
   - Запустіть Microsoft Excel.
   - Оберіть **Файл -> Відкрити** та виберіть `data/test.csv`.
   - *Важливо:* Якщо Excel некоректно відображає символи (наприклад, українські літери), використовуйте функцію **Дані -> Отримати дані -> З тексту/CSV** та оберіть кодування **UTF-8**.
3. **Редагування**:
   - Виправте існуючі або додайте нові рядки.
   - Переконайтеся, що колонки `Number`, `Question` та `Answer` залишаються на своїх місцях.
   - Ви можете використовувати формули LaTeX у колонці `Answer`, огортаючи їх символами `$` (наприклад, `$E=mc^2$`).
4. **Експорт у CSV**:
   - Оберіть **Файл -> Зберегти як**.
   - У списку форматів обов'язково оберіть **CSV (розділювач - кома) (*.csv)** або **CSV UTF-8 (розділювач - кома) (*.csv)**.
5. **Заміна**:
   - Збережіть файл під назвою `test.csv` у папку `data`, замінивши старий файл.
   - Додаток автоматично оновить дані при наступному завантаженні сторінки (завдяки кешуванню Streamlit може знадобитися натиснути "R" в інтерфейсі або перезапустити сервер).

## ⚙️ Конфігурація
Ви можете змінити шлях до файлу з даними у файлі `config.yaml`:
```yaml
data:
  path: "data/test.csv"
```
